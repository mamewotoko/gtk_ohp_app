gtk_ohp_app
====================

Transparent window

![](doc/sample.png)

Prepare
-------

clone this source

```
git clone --recursive https://github.com/mamewotoko/gtk_ohp_app.git
```

install GTK

```
sh setup.sh
```


Build
------

```
make
```

Run
----

```
./mini
```
